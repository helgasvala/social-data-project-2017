The Project

Our project aims at exploring the status of women in the world of movies. We want to analyze how they appear, in acting, directing and writing. We want to see how the ratings, as well as the genre of their movies are compared to those of men. We also use term frequency - inverse document frequency analysis to see which words are the most likely to be used in the user reviews, and check the sentiment of the reviews.

Getting the data for our project was the biggest challenge. It was very hard to acquire some user reviews, which we considered the biggest part of the task, but finally we were able to get them from The Movie Database. We get lists of movies made in specific years from Wikipedia, information about the movies returned from The Open Movie Database (an imdb API), and to determine which gender the people on our list are, we (partly) use a name gender API which only allows 1000 requests a day. Some harder names were also gender-determined by hand.
