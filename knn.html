The Project

In this section, we perform KNN-classification (K-nearest neighbours). The input in the algorithm consists of the k closest training-examples we have. The output is a class membership, where an object is assigned to the class most common among its k nearest neighbours. We classify examples based on if the dropout rate in the neighbours is low (under 5%) medium (5-25%) or high (over 25%). We use all the data we have, with every year and every planned study length.
